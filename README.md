Autism Spectrum Disorder (ASD) often affects a child's ability to communicate needs and regulate emotions, making continuous supervision necessary for safety and well-being. This project presents AutiGuard, a smart wearable watch prototype designed to assist autistic children by monitoring physiological signals, detecting emotional distress, and enabling simple communication with caregivers.  The system is built using an ESP32 microcontroller integrated with multiple sensors including MAX30100 pulse oximeter for heart rate monitoring, MPU6050 accelerometer for fall detection, NEO-6M GPS module for real-time location tracking, and a sound sensor to detect crying or distress sounds. Additionally, three push buttons are incorporated to allow the child to easily communicate basic needs such as hunger, thirst, and restroom requirement through pictorial indicators.  The wearable device continuously collects sensor data and transmits it to a cloud-based web application through Wi-Fi when the child is near parents or teachers. If Wi-Fi connectivity is unavailable, the system automatically switches to GSM communication using the SIM800L module, ensuring uninterrupted monitoring even when the child moves away from the nearby network.  The cloud platform processes the transmitted data and displays alerts, heart rate information, and location updates for caregivers. Emergency conditions such as panic (high heart rate), crying detection, falls, or button-triggered requests generate immediate notifications.  The proposed system aims to enhance child safety, emotional monitoring, and communication assistance, thereby supporting caregivers in providing timely help and improving the quality of care for autistic children.
